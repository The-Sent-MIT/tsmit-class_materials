# User List Next.js
- 서버 사이드 렌더링를 사용하여 API 데이터를 가져오고 화면에 표시한다.
- 유저 리스트에서 유저 검색 기능을 구현한다.
- users/{id} URL을 구현한다.
- 유저 리스트에서 유저 이름을 클릭하면 유저 디테일 페이지로 이동한다.

### 과제 시나리오
1. 서버 사이드 렌더링를 사용하여 API 호출 후 메인 컴포넌트에 유저 리스트 작성.
2. 검색창에서 유저를 검색/필터링 할 수 있음.
3. 유저 리스트에서 이름 클릭 시 해당 유저의 상세 정보 페이지로 이동.
4. 상세정보 페이지에서는 유저의 상세 정보와 홈으로 돌아가는 버튼 구현.

### URL
- 모든 유저를 불러오기
  - GET: `https://jsonplaceholder.typicode.com/users`
- 아이디를 이용해 특정 유저의 세부 정보 불러오기
  - GET: `https://jsonplaceholder.typicode.com/users/${id}`
